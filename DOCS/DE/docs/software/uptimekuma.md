# Uptime Kuma
Uptime Kuma ist ein einfaches Monitoring Tool. Damit kann man überwachen welche Dienste und Geräte ordnungsgemäß laufen und ggf. eine Mitteilung über eine Störung versenden.
Das ist durchaus praktisch.
